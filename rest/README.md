# The Culinary Atelier - Restaurant Landing Page

Elegant restaurant landing page for a fine dining concept. The page includes a full-screen hero, restaurant story section, signature menu cards, image gallery, reservation form, and footer.

## Structure

- `index.html` - page markup and Tailwind utility layout
- `styles.css` - custom typography, body styles, and page-specific styling
- `script.js` - page-level JavaScript entry point
- `tailwind.config.js` - Tailwind theme colors, fonts, and radius settings
