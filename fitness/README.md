# Kinetic Edge - Fitness Landing Page

Dark, high-contrast fitness landing page for an elite training studio. The page focuses on strong visual hierarchy, bold typography, program cards, trainer profiles, testimonials, pricing tiers, and a final email CTA.

## Structure

- `index.html` - page markup and Tailwind utility layout
- `styles.css` - custom page styles, reveal animation states, and header effects
- `script.js` - scroll reveal animation and sticky header behavior
- `tailwind.config.js` - Tailwind theme colors, fonts, and radius settings
