# Neon Logic - IT Landing Page

Modern digital agency landing page for a technology and product design studio. The page includes a hero section, service cards, project inquiry form, glassmorphism panels, and a compact footer.

## Structure

- `index.html` - page markup and Tailwind utility layout
- `styles.css` - custom visual effects, gradients, glass panels, and icon styles
- `script.js` - page-level JavaScript entry point
- `tailwind.config.js` - Tailwind theme colors, fonts, and radius settings
